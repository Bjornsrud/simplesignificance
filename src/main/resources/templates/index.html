Welcome to Simple Significance

A Simple way to do basic hypothesis testing





How should the CSV file be formatted?

The file should contain two columns, one for each group (e.g. control group and experimental group).
The first row may optionally contain a project title or description, in which case it should only have one cell.
The second row can contain headers like ControlGroup,ExperimentalGroup (optional but recommended).
Each subsequent row should contain numeric values. Blank cells are allowed and will be ignored.

You can create a file like this in Microsoft Excel or LibreOffice Calc, then choose “Save As...” and select “CSV (Comma delimited)” as the file type.
Alternatively, you can create the .csv manually in a simple text editor.

Which statistical test should I choose?

T-Test
Use when comparing the means of two separate groups (e.g., control vs. experimental).
Assumes normally distributed data and similar variance.

Welch's T-Test
Like the regular T-Test, but allows for unequal variances between groups.
Recommended if one group has much more variability than the other.

Paired T-Test
Use when comparing two related sets of scores (e.g., before and after an intervention for the same individuals).
Requires normal distribution of the differences.

Mann-Whitney U Test
A non-parametric alternative to the T-Test.
Use when your two groups are independent, but the data is not normally distributed or contains outliers.

Wilcoxon Signed-Rank Test
A non-parametric alternative to the Paired T-Test.
Use when comparing related scores that are not normally distributed.

ANOVA
Use when comparing the means of three or more groups.
Assumes normal distribution and equal variances.
For two groups, consider using a T-Test if the data is normally distributed; otherwise, consider a non-parametric alternative like Mann-Whitney.
